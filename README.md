# VirtualDub
VirtualDub is a video capture/processing utility for 32-bit and 64-bit Windows platforms (98/ME/NT4/2000/XP/Vista/7).

Initially hosted at http://virtualdub.com/ and developed by Phareon, but long since abandoned. This repo is for back-up purposes only, should the original creator wish to pick it up in the future, or should somebody else wish to fork and continue to develop it.
